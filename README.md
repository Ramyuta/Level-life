# Level-Life

Level-Life は、ゲーム感覚で毎日の成長を追跡できる Next.js 製のデイリー成長トラッカーです。タスクの達成状況と獲得 XP を可視化し、レベルアップやストリーク機能でモチベーションを高めます。

## 主な機能

- ✅ タスクの追加・削除・完了チェック
- ✅ タスク完了時の XP 獲得とレベルアップ
- ✅ ストリークと最長ストリークの自動計測
- ✅ モバイルでも見やすいレスポンシブ UI
- ✅ ローカルストレージによるデータ永続化

## セットアップ

```bash
pnpm install # または npm install / yarn install
pnpm dev     # または npm run dev
```

ブラウザで `http://localhost:3000` を開くとアプリにアクセスできます。

## 使用技術

- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
